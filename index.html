How to apply a font in Word after converting

  1. Download and install the font on your computer (restart Word if already open).
  2. Open the converted .docx in Word.
  3. Press Ctrl+A (or ⌘+A) to select all text.
  4. In the Home tab → Font dropdown, choose the Hebrew font (e.g. Stam Ashkenaz CLM).
  5. Hebrew text renders in the new font immediately. English runs may need to be reset separately.
  6. For best results with nikud and trup, use Stam Ashkenaz CLM or Ezra SIL.
